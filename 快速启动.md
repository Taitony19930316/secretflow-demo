## 第一次启动

1. **安装/更新依赖**（如果配置文件有更新）：
```bash
cd /Users/tailunyu/Desktop/sf/secretflow-demo
npm install
```

2. **启动开发服务器**：
```bash
npm run docs:dev
```

3. **访问网站**：
打开浏览器访问：http://localhost:8080

## 常见问题

**Q: 端口8080被占用怎么办？**
A: VuePress会自动选择其他端口（如8081），查看终端输出的实际端口号即可。

**Q: 修改文档后看不到变化？**
A: VuePress支持热重载，保存文件后自动刷新。如果没有刷新，手动刷新浏览器页面。
